# Customer Support Operations Dashboard

![Dashboard Preview](customer-support-dashboard.png)

## Overview
This project analyzes customer support ticket data to identify trends in ticket status, issue types, and resolution times. It demonstrates how operational analytics can reveal workload distribution and team efficiency.

## Objectives
- Track ticket volume by status
- Identify the most frequent issue types
- Measure average resolution time across categories
- Visualize insights in a clear dashboard format

## Tools Used
- Google Sheets
- Pivot Tables
- Data Visualization (Charts)

## Key Insights
- Majority of tickets are resolved, showing overall efficiency
- Technical issues make up the largest share of tickets
- Some categories have significantly longer resolution times, highlighting potential bottlenecks
